event

Functions can be assigned to events, in this case, when the load event is triggered (when the page finishes loading) the function, loaded(), will be run.

Also, when ever you move your mouse the pageX/Y of the move event will be put in the tags below. The pageX/Y property has been corrected with Spark so that it can be used in Internet Explorer and all the other browsers.

focus

Focus can be applied to elements that allow focus, such as an input with a type of 'text'

Even though there is an input before it, the second one has focus, this is the focus function in action.

opacity

The opacity of an element can be retreived, set or set over a period of time, the example for this is the whole page fading in on load.
