Spark by Wolfy87

A lightweight yet powerful JavaScript library.

Spark is an easy to use, lightweight JavaScript library. It features animation of size, location and opacity. It can also perform AJAX calls. It is always growing because it is my main project.

Dependencies

Spark uses the Sizzle CSS selector engine, this is included inside Spark.

Install

To get spark working all you have to do is download and include the file. For the best results make all of your code execute after the document load event is fired like so.
